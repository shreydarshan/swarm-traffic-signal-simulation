# Swarm-Based Traffic Signal Simulation

This project presents a simulation of an intelligent traffic control system using swarm intelligence. The system dynamically adjusts traffic signals based on vehicle density and provides priority to emergency and VIP vehicles.

## Overview

Traditional traffic systems use fixed timers which often lead to congestion and delays. This project introduces a decentralized approach where each intersection acts as an independent intelligent agent. These agents communicate with each other and optimize traffic flow in real time.

## Features

- Dynamic traffic signal control  
- Swarm-based decentralized decision making  
- Emergency vehicle preemption (green wave system)  
- Real-time simulation using Python  

## Working

Each traffic signal:
- Detects vehicle density on each road  
- Communicates with nearby signals  
- Adjusts signal timing dynamically  
- Gives priority to emergency vehicles  

## Technologies Used

- Python  
- Pygame  
- NumPy  
- Matplotlib  

## How to Run

1. Install dependencies:pip install -r requirements.txt 
2. Run the simulation:python main.py

## Results

- Reduced traffic waiting time  
- Improved traffic flow  
- Faster movement for emergency vehicles
